


# Driving School Web Application

## Overview
This Node.js web application is designed for a driving school, featuring user registration, login, bookings, and account management. It uses MongoDB as the database.

## Configuration
1. **Node.js**: Ensure Node.js is installed.
2. **MongoDB**: Install MongoDB and ensure it is running on your system.
3. **Dependencies**: After cloning the repository, run `npm install` to install required packages.

## Deployment
1. **Start MongoDB**: Run the MongoDB service.
2. **Start the Application**: Run `node server.js` in the project directory. The application will be hosted on `http://localhost:3000`.
3. **Access the Web Application**: Open a web browser and navigate to `http://localhost:3000`.


Features
Web Pages: Home, About Us, Courses, Bookings, Account Management, and Location Map.
User Registration: New users can register.
User Login: Existing users can log in.
Booking Management: Users can make and manage bookings.
Account Management: Users can view and manage their accounts.
Map Integration: Provides location details of the school.
